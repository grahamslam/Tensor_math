# Methodology Reference

**Scope:** Computational methods used in the Ramsey research (R(3,3), R(3,4), R(3,5), R(4,4), R(5,5)).

---

## 1. Exact Computations

### 1.1 Extension Resistance (ER) -- Exhaustive

For small n (n <= 16), ER is computed exactly by exhaustive enumeration of all 2^n connection vectors. For each candidate vector c in {0,1}^n, the (n+1)-vertex graph G+c is formed and the number of new-vertex violations (r-cliques + s-independent-sets involving the new vertex) is counted. ER is the minimum over all 2^n vectors.

- **Script:** `scripts/analyze_er_small_ramsey.py`, function `compute_er(..., method='exhaustive')`
- **Applies to:** R(3,3) n=4-5, R(3,4) n=6-8, R(3,5) n=10-13 (ER computation), R(4,4) n=14-16 (ER computation)
- **Cost:** O(2^n * C(n, r-1)) per graph

### 1.2 Clique / Independent-Set Counting

All clique and independent-set counts (triangles, K4, I4, K5, I5) are computed by exhaustive enumeration of all k-subsets via `itertools.combinations`. No approximations. The complement graph is constructed explicitly (C = 1 - A with diagonal zeroed) and cliques in the complement serve as independent sets in the original.

- **Functions:** `count_cliques()`, `has_clique()`, `count_new_violations()` in the analysis scripts

### 1.3 Structural Properties

For each graph, the following are computed exactly:
- Edge count, degree distribution, degree variance
- Triangle counts (G and complement) via matrix trace: tri = trace(A^3) / 6
- K4 and I4 counts via exhaustive 4-subset enumeration
- Eigenvalues via `numpy.linalg.eigvalsh` (symmetric eigenvalue decomposition)
- Spectral gap (lambda_1 - lambda_2), Hoffman bound

### 1.4 Avoiding Graph Enumeration (Small n)

For n <= 7, all 2^(n choose 2) labeled graphs are generated and tested for R(r,s) avoidance by checking for r-cliques and s-independent-sets. No isomorphism filtering is applied; the enumeration is over labeled graphs.

- **Script:** `scripts/analyze_er_small_ramsey.py`, function `generate_all_graphs(n)`
- **Applies to:** R(3,3) n=4-5 (all 2^6=64 and 2^10=1024 graphs), R(3,4) n=6-7 (all 2^15=32768 and 2^21=2097152 graphs)

## 2. SA-Derived Upper Bounds

### 2.1 ER Upper Bounds for R(5,5) n=42

ER values for the 656 R(5,5,42)-graphs are **upper bounds** on true ER, computed via simulated annealing on the 42-bit connection vector.

**Parameters (analyze_extension.py, `sa_extension()`):**
- Restarts: 1 per graph (original) + 1 per graph (complement) = 1 restart each
- Steps per restart: 5,000
- Initial temperature: T_0 = 1.0
- Cooling: multiplicative, rate = 1.0 - 5.0/steps = 0.999
- Final temperature floor: max(T, 0.005)
- Acceptance: Metropolis (accept if dE <= 0, else accept with probability exp(-dE/T))
- Initial connection vector: random 50/50
- Move: flip one random bit of the 42-bit connection vector
- Random seed: unseeded (`np.random.RandomState()` with no fixed seed)

**Output:** `results/extension_analysis.json`

**Important caveat:** With only 1 SA restart and 5,000 steps per 42-dimensional search space, these are loose upper bounds. The reported ER=2 for Graph 41 means the true ER is at most 2 (and at least 1, since ER=0 would imply extendability to n=43, contradicting the known bound).

### 2.2 ER Upper Bounds for Larger Small-Ramsey Cases

For R(4,4) n=17, ER is computed via SA rather than exhaustively (2^17 = 131,072 would be feasible but the script uses the SA path for n > 16).

**Parameters (analyze_er_small_ramsey.py, `compute_er(..., method='sa')`):**
- Restarts: 3
- Steps per restart: 5,000
- Initial temperature: T_0 = 1.0
- Cooling: multiplicative, rate = 1.0 - 5.0/steps = 0.999
- Final temperature floor: max(T, 0.005)
- Random seed: unseeded

## 3. Graph Construction Methods

### 3.1 Exhaustive Enumeration (n <= 7)

All labeled graphs on n vertices are generated by iterating over all 2^(n choose 2) edge masks. Each is tested for R(r,s) avoidance. No isomorphism reduction.

### 3.2 SA-Based Construction (n > 7, small Ramsey)

For R(3,5) and R(4,4) at larger n, avoiding graphs are found by SA that minimizes total violations (r-cliques + s-independent-sets) to zero.

**Parameters (analyze_er_small_ramsey.py, `find_avoiding_graphs_sa()`):**
- Target: 500 distinct avoiding graphs per (r, s, n) case
- Max SA restarts: 3,000 (but terminates early once 500 graphs found)
- Steps per restart: 8,000 (in the call); internally 5,000 (in the function default)
  - Note: the `__main__` block passes `steps=8000` to `find_avoiding_graphs_sa()` but the function signature shows `steps=5000` as default; the call overrides with 8,000.
- Initial temperature: T_0 = 2.0
- Cooling: geometric, rate = (0.01/2.0)^(1/steps), cooling from 2.0 to 0.01
- Move: flip one random edge
- Initial graph: Erdos-Renyi G(n, 0.5)
- Random seed: fixed at 42 (`np.random.RandomState(42)`)
- Deduplication: by adjacency matrix bytes (labeled graph identity)

### 3.3 Circulant Construction (R(5,5))

For R(5,5) at n=38-41, the search is restricted to circulant graphs on Z_n defined by a difference set S. Edge (u,v) exists iff min(|u-v|, n-|u-v|) is in S. Search space is 2^(floor(n/2)) instead of 2^(n(n-1)/2).

**Pipeline:**
1. **Phase 1 -- Random sampling:** Sample random difference sets, evaluate full violations. Found pure circulant avoiders at n=40 and n=41.
2. **Phase 2 -- Difference-set SA:** SA over the difference set bits, minimizing violations.
3. **Phase 3 -- Edge SA:** Start from the best circulant, break symmetry by flipping individual edges. Critical parameter: low initial temperature (T=0.3) to preserve circulant structure.

**Scripts:** `scripts/r55_search.py` and related search scripts (see R55_search_report.md)

### 3.4 Paley Graphs

Paley(p) for prime p = 1 mod 4: vertices are elements of F_p, edge (a,b) iff (a-b) is a quadratic residue. Self-complementary. Used as seeds and as verified avoiders (Paley(37) for R(5,5)).

### 3.5 McKay's R(5,5,42) Dataset

328 graphs in graph6 format (`results/r55_42some.g6`). Each graph plus its complement yields 656 distinct R(5,5,42)-avoiding graphs. Source: Brendan McKay's collection of all known R(5,5)-critical graphs on 42 vertices.

### 3.6 Exoo's n=42 Construction

Exoo (1989): start from the cyclic group on 43 vertices, delete one vertex, flip 16 specific edges. Produces a non-circulant R(5,5,42)-avoider with degree range 19-22. Verified independently by our scripts.

## 4. Verification Procedures

### 4.1 Ramsey Avoidance Verification

Every claimed R(r,s)-avoiding graph is verified by:
1. Exhaustive enumeration of all C(n, r) subsets, checking for r-cliques in G
2. Exhaustive enumeration of all C(n, s) subsets, checking for s-cliques in the complement (= s-independent-sets in G)

Both counts must be zero. No sampling, no spectral bounds -- full enumeration only.

- **Function:** `is_ramsey_avoiding()` checks for any clique; `count_violations()` returns the exact count

### 4.2 Extension Resistance Verification

For exhaustive ER (n <= 16): all 2^n connection vectors are tested. The minimum violation count is exact.

For SA ER: the reported value is an upper bound. ER=0 results are exact (if SA finds a zero-violation extension, that extension is verified by the clique-counting code). ER > 0 results mean "no zero-violation extension was found by SA" but do not prove non-extendability.

### 4.3 Barrier Analysis (Graph 41)

For Graph 41 (ER=2), deeper verification was performed:
- The 2 remaining violations are specific K5s through identified 4-cliques
- Multi-edge swap analysis: all 26,335 pairs of up to 3 simultaneous edge modifications tested exhaustively
- **Script:** `scripts/analyze_barrier.py`

## 6. Reproducibility

### 6.1 Deterministic Components

- **Exhaustive graph enumeration** (n <= 7): fully deterministic, iterates over all 2^(n choose 2) edge masks
- **Exhaustive ER** (n <= 16): fully deterministic, iterates over all 2^n connection vectors
- **Clique counting:** fully deterministic, iterates over all k-subsets
- **Eigenvalue computation:** deterministic (numpy.linalg.eigvalsh)

### 6.2 Stochastic Components

- **SA for ER upper bounds** (`analyze_extension.py`): unseeded RNG. Results are not exactly reproducible across runs, but the upper-bound property holds regardless. Different runs may find tighter or looser bounds.
- **SA for avoiding graph construction** (`analyze_er_small_ramsey.py`, `find_avoiding_graphs_sa()`): seed = 42. Reproducible given the same numpy version and platform.
- **SA for R(5,5) graph construction** (`r55_search.py` and related): various seeds used across different search phases.
